# Agency Research

Интерактивный отчет по работе агентств с клиентами Авто.ру за Q1 2026.

## Что внутри

- `index.html` - основной самодостаточный дашборд с вкладками.
- `docs/agency_classifieds_research_presentation.html` - отдельная HTML-презентация по агентствам.
- `tools/build_agency_dashboard.py` - скрипт, которым собирался дашборд.

## Разделы дашборда

- Дашборд: выручка, платные услуги, агентства, регионы, ГК и дилеры.
- Агентства: презентационное сравнение CarCopy, 4 Пикселя и ТАК / Tandem Group.
- Анализ выручки агентств: оценка выручки от сопровождения.
- Расчет потенциала агентства Авто.ру: сценарии роста и перехвата клиентов.
- Экономическая модель: ФОТ, загрузка команды и план продаж.

## Как открыть

Откройте `index.html` в браузере. Файл не требует сервера и внешних зависимостей.

Если нужен общий веб-адрес для коллег, включите GitHub Pages:

1. Откройте настройки репозитория.
2. Перейдите в `Pages`.
3. В разделе `Build and deployment` выберите `Deploy from a branch`.
4. Укажите ветку `main` и папку `/root`.
5. После публикации GitHub покажет ссылку на отчет.

## Источники

Данные дашборда рассчитаны по выгрузке DataLens за Q1 2026. Презентация по агентствам собрана по открытым источникам и переданным вводным.
